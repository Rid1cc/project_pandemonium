
# Pandemonium 

Game "Hacking Simulator" based on C++ and raylib combo.




## Dev Installation

- install cmake
- install raylib (via brew, or include to cmake manually)
- clone repository
- open repository master folder in terminal

```bash
  cd build
  cmake ..
  make
  ./pandemonium
```
### Tested on:
- MacOS >=15.0
- Debian based linux
